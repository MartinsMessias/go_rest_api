# Go Rest API

#### Structure

```bash
├── controllers
│ └── controllers.go
├── database
│ └── db.go
├── dev.env
├── go.mod
├── go.sum
├── main.go
├── middleware
│ └── middleware.go
├── models
│ ├── config.go
│ └── personalities.go
└── routes
     └── routes.go
```

## Stack used

**Backend:** Go

**Database:** Postgres

## Environment variables

To run this project, you will need to add the following environment variables to your .env

`dbhost`

`dbport`

`dbuser`

`dbname`

`dbpassword`
